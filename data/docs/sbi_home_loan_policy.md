# SBI Housing Finance Policy Update: Green Home Loans
**Circular No:** HL/2024/05
**Subject:** Introduction of "SBI Green Home Loan" for Eco-friendly Projects.

## 1. Product Overview
The "SBI Green Home Loan" is designed to encourage customers to purchase properties in Green Building Council (IGBC/LEED) certified projects using eco-friendly materials detailed in the project report.

## 2. Key Features
- **Interest Concession:** 0.25% (25 bps) discount on the standard card rate.
- **Processing Fee:** 50% waiver on processing charges.
- **Max Loan Amount:** No cap (subject to LTV norms).

## 3. Eligibility
- Project must have valid Gold or Platinum rating from IGBC/LEED.
